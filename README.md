## Quick start
- Clone the repo: `git clone https://github.com/nitinsahu-sahu/tick-m-events.git`
- Recommended: `Node.js v20.x`
- **Install:** `npm i` or `yarn install`
- **Start:** `npm run dev` or `yarn dev`
- Open browser: `http://localhost:8000`
